# Boston House Price Prediction

## Overview
This project predicts Boston house prices using Machine Learning techniques.

## Dataset
- Boston Housing Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation

## Model Used
- Random Forest Regressor

## Results
- MAE: 2.067691176470588
- RMSE: 2.943098899103114
- R² Score: 0.8818848454747603
